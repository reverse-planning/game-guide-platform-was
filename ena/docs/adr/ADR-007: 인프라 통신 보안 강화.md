# ADR-007: 인프라 통신 보안 강화 (HTTPS)

## 상태
Accepted

## 컨텍스트
인증 토큰(JWT)과 개인정보가 네트워크 구간에서 탈취되는 것을 방지하기 위한 보안 프로토콜 적용이 필수적이었습니다.

## 결정
모든 API 통신에 **HTTPS(TLS)** 프로토콜을 의무적으로 적용합니다.

## 근거
- **데이터 암호화**: 종단 간 암호화를 통해 패킷 스니핑 공격으로부터 데이터를 보호합니다.
- **Secure 쿠키**: 우리가 채택한 Refresh Token용 쿠키 보안 옵션(`Secure`)을 사용하기 위한 전제 조건입니다.

## 결과
- 보안 사고 위험을 낮추고 서비스의 신뢰도를 확보했습니다.
